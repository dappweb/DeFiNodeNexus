3) 验收标准（Go / No-Go）

Go 标准

  • 前端可正常启动，关键页面可访问
  • E2E 全量通过（10/10）
  • Round3 在 1~5 次重试内可达成功退出（EXIT:0）
  • 关键交易链路可复现：购买、领取、提现、Swap

No-Go 标准

  • E2E 关键用例持续失败
  • 管理员参数写链失败且不可恢复
  • Round3 持续失败并定位为业务逻辑回退（非网络）
