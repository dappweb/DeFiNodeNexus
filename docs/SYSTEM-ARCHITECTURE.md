## 页面详细流程

### 1️⃣ HomePage - 首页

```
┌─────────────────────────────────┐
│        HomePage 组件             │
└─────┬───────────────────────────┘
      │
      ├─➀ 加载动态公告
      │   GET /api/announcements
      │   └─> UI 显示公告列表
      │
      ├─➁ 加载用户概览数据
      │   ├─ 获取用户信息
      │   ├─ 获取 NFT 数量
      │   └─ 获取待领取分红
      │
      ├─➂ 显示新用户引导
      │   ├─ 连接钱包
      │   ├─ 切换网络
      │   ├─ 首次购买
      │   └─ 绑定推荐人
      │
      └─➃ 显示系统统计
          ├─ 在线用户
          ├─ 总收益
          └─ TVL
```

### 2️⃣ NodesPage - NFT 管理

```
┌─────────────────────────────────┐
│        NodesPage 组件            │
└─────┬───────────────────────────┘
      │
      ├─➀ 加载用户 NFTA 卡牌
      │   └─ nexus.getUserNFTA()
      │       ├─ 数量、等级、收益
      │       └─ 可转卡、可升级
      │
      ├─➁ 加载用户 NFTB 卡牌
      │   └─ nexus.getUserNFTB()
      │       ├─ 数量、等级、收益
      │       └─ 可转卡、可升级
      │
      ├─➂ 购买新 NFT 操作
      │   ├─ 选择等级
      │   ├─ 选择支付币种 (USDT/TOF)
      │   ├─ 批准代币支出
      │   └─ 执行 mint 交易
      │
      ├─➃ 转卡操作
      │   ├─ 选择转卡对象
      │   └─ 执行转卡交易
      │
      └─➄ 收益显示
          ├─ 日收益
          ├─ 月收益
          └─ 历史分红记录
```

### 3️⃣ SwapPage - 交换池

```
┌─────────────────────────────────┐
│        SwapPage 组件             │
└─────┬───────────────────────────┘
      │
      ├─➀ 获取池状态
      │   ├─ TOT 价格
      │   ├─ 流动性总额
      │   ├─ 下次通缺时间
      │   └─ 手续费信息
      │
      ├─➁ 交换 TOT 为 USDT
      │   ├─ 输入金额
      │   ├─ 计算输出数量
      │   ├─ 批准 TOT 支出
      │   └─ 执行 swap 交易
      │
      ├─➂ 交换 USDT 为 TOT
      │   ├─ 输入金额
      │   ├─ 计算输出数量
      │   ├─ 批准 USDT 支出
      │   └─ 执行 swap 交易
      │
      └─➃ 显示池历史
          ├─ 价格曲线
          ├─ 交易量
          └─ 通缺记录
```

## 数据流图

### 用户操作流

```
用户操作
   │
   ├─➀ 购买 NFT
   │   ├─ 选择等级
   │   ├─ 选择币种 (USDT/TOF)
   │   ├─ 批准代币 (approve)
   │   ├─ 执行购买 (mint)
   │   ├─ 链上验证
   │   └─ UI 刷新
   │
   ├─➁ 转卡操作
   │   ├─ 获取持卡列表
   │   ├─ 选择转卡对象
   │   ├─ 执行转卡 (transferNFT)
   │   ├─ 链上验证
   │   └─ UI 刷新
   │
   ├─➂ 交换代币
   │   ├─ 输入数量
   │   ├─ 计算输出
   │   ├─ 批准支出 (approve)
   │   ├─ 执行交换 (swap)
   │   ├─ 链上验证
   │   └─ UI 刷新
   │
   ├─➃ 领取分红
   │   ├─ 查询待领取
   │   ├─ 执行领取 (claim)
   │   ├─ 链上验证
   │   └─ UI 刷新
   │
   └─➄ 绑定推荐人
       ├─ 验证推荐人
       ├─ 签名绑定
       ├─ 链上验证 (一次性)
       └─ UI 更新
```

### Keeper 执行流

```
定时触发
   │
   ▼
检查文件锁 (防重入)
   │
   ├─ 锁定 ✓
   │   ▼
   │  执行任务
   │   │
   │   ├─➀ 检查通缩条件
   │   │   └─ 是否满足时间 + 条件
   │   │
   │   ├─➁ 执行 deflate()
   │   │   └─ gas 计算、签名、上链
   │   │
   │   ├─➂ 检查 TOT 分红
   │   │   └─ 池余额 >= 阈值
   │   │
   │   ├─➃ 执行 TOT 分配
   │   │   └─ 调用 forceDistribute()
   │   │
   │   ├─➄ 检查 USDT 分红
   │   │   └─ 池余额 >= 阈值
   │   │
   │   ├─➅ 执行 USDT 分配
   │   │   └─ 调用 forceDistribute()
   │   │
   │   └─➇ 记录状态文件
   │       └─ latest-run.json
   │
   └─ 锁定 ✗
       └─ 跳过 (已有实例运行)

状态文件 (runtime/keeper/latest-run.json)
   │
   ├─ timestamp
   ├─ tasks_executed[]
   │   ├─ name
   │   ├─ executed
   │   ├─ hash
   │   └─ gas_used
   └─ errors
```

## 权限和访问控制

```
┌─────────────────────────────────────┐
│        用户认证路径                  │
├─────────────────────────────────────┤
│                                      │
│  连接钱包                            │
│     │                                │
│     └─> 获取用户地址                 │
│            │                         │
│            └─> 检查 Owner 权限       │
│                   │                  │
│           ┌───────┴───────┐          │
│           │               │          │
│        是 (Owner)     不是 (普通用户)│
│           │               │          │
│           ▼               ▼          │
│   ┌────────────┐    ┌──────────────┐│
│   │显示 Admin  │    │只显示 5 页面 ││
│   │页面        │    │隐藏 Admin    ││
│   │所有功能    │    │面板          ││
│   │都可用      │    └──────────────┘│
│   └────────────┘                    │
│                                      │
└─────────────────────────────────────┘

检查方式：
1. 链上查询: reader.owner() (准确但慢)
2. 环境变量: NEXT_PUBLIC_CONTRACT_OWNER (快但手动)
3. 两者结合: 先检查环保变量，再验证链上
```
